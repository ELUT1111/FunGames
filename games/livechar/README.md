# 活字灵境 · Living Type

> 一个融合**活字印刷术、水墨书法与蒸汽朋克美学**的 3D 动作解谜游戏原型。
> 核心:拥有自我意识的活字灵体 ——「凡所书写,皆成真实」。

纯前端实现(Three.js,本地依赖),**双击 `index.html` 即可游玩**,无需安装任何东西。
推荐使用 Chrome / Edge 浏览器。

---

## 世界观与流程

开局是无边的纯黑虚空,只有微光活字悬浮。你是一枚名为「人」的活字。

1. **收集世界活字**(跟随光柱):
   - 「山」(西北)→ 文字隆起为层峦叠嶂
   - 「水」(东北)→ 墨线流淌成粼粼大河
   - 「树」(西南)→ 笔画抽枝,字海成林
   - 「城」(东南,需渡裂谷)→ 蒸汽朋克古城拔地而起
2. 每收集一字,**主角同步进化**:字 → 词语 → 词组 → 句子 → **活书形态**
3. 集齐四字后,蚀文之主**「噬」**苏醒 —— 击败它,完成原型通关
4. 隐藏收集:**三枚古印**(主峰之巅 / 大河彼岸 / 城中钟楼),提升墨量与墨魂上限

| 组合 | 效果 |
|---|---|
| 火 + 箭 | 追踪火焰箭 ×5 |
| 冰 + 墙 | 寒冰屏障 ×3(阻挡敌人) |
| 时 + 缓 | 时间减速 ×2(全场缓滞) |
| 隐 + 身 | 隐形潜行 ×2(脱离仇恨) |
| 风 + 刃 | 风刃刃舞 ×4(扇形五连·穿透) |
| 雷 + 落 | 天雷落字 ×3(轰击准星落点) |
| 火 + 雨 | 焚天火雨 ×2(火字成雨倾泻,火字多用!) |
| 影 + 身 | 墨影分身 ×2(嘲讽敌人,终乃自爆,身字多用!) |
| 愈 + 心 | 妙笔回春 ×3(回复墨魂与墨量) |
| 木 + 石 | 「桥」字(在裂谷标记处按 F 书写成桥) |
| 金 + 木 | 「钥」字(开启文匣宝箱,木字多用!) |

## 场景交互(每个字都有用处)

- **金矿石**(山区,F 提炼)→「金」→ 合成「钥」→ 开启四处**文匣宝箱**(稀有活字 + 墨)
- **四座古碑**(各区域附近,F 拓读):每碑授一枚主题活字(刃/雨/心/雷)并赠墨
- **河中字鱼**:水域成真后,「鱼」字不时跃出河面,捕之得墨
- **古木 / 巨石**:抽象化为「木」「石」,既是合成材料,拆树还会引发文字洪流伤敌

活字来源:虚空中散落的发光小字、击败墨魉(魉/煞)掉落(12% 出稀有字)、古碑、文匣。

## 非强制性支线(J 键打开书阁图鉴)

主线只需集齐山水树城并击败「噬」;以下内容全部可跳过,但提供永久成长回报:

- **《活字谣》残诗拾遗**:六句残句藏于地图深处(紫色光柱;书阁显示地点线索,
  其中「齿轮之上」需要「轻」字跳跃)。集齐成诗 → 墨魂与墨量上限大幅提升
- **试炼三坛**(F 启坛,失败可重试):
  - 「斩」之坛(红):60 秒斩灭 8 只试炼墨魉 → 墨痕斩伤害 22→32
  - 「行」之坛(青):40 秒依次穿过 6 道字环 → 墨遁冷却减半、耗墨 8→5
  - 「守」之坛(金):符阵内坚守 30 秒 → 反墨盾持续翻倍、反伤 30→45
- **精英狂魉**:世界苏醒后随机出没的强敌,必掉 3 枚稀有活字
- **十项成就**:初啼 / 渡谷 / 拓碑人 / 开匣师 / 鱼跃 / 千字文 / 诗心 / 三坛圆满 / 屠狂 / 书成
