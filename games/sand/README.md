# 沙漏行者 · The Hourglass Wanderer

一个融合**沙之美学、时间哲学与记忆叙事**的 3D 动作冒险解谜游戏原型。

你是一具几乎空无一物的沙漏残躯，行走在吞噬了一切文明的金色沙海之上。
找回散落世界各处的记忆之沙，决定哪些记忆值得背负，哪些该永远埋葬——
你的每一次选择，都将重塑你的形态，以及这个世界的结局。

## 运行方式

**方式一（推荐）**：任意静态服务器

```bash
cd sand
npx -y http-server -p 8080
# 浏览器打开 http://localhost:8080
```

**方式二**：直接双击 `index.html`（采用经典脚本加载，file:// 协议亦可运行）

> 需要联网：Three.js r147 由 jsDelivr CDN 加载（约 600KB，仅首次）。
> 推荐 Chrome / Edge，桌面端 + 鼠标键盘。

## 操作

| 按键 | 功能 |
|------|------|
| `W A S D` | 移动 |
| 鼠标 | 视角（点击画面锁定指针）/ 滚轮调整视距 |
| `空格` | 跳跃（「时之完体」形态可二段跳） |
| `F` | **翻转沙漏** —— 逆转周围的时间流动 |
| `E` | 与虚影交谈 / 圣坛交互 |
| `左键` | 沙之脉冲（驱散沙魇与沙弹，需「沙肤」以上形态） |
| `Esc` | 暂停 |

## 核心系统

- **沙漏行者**：双玻璃球躯体 + 内部实时流沙。随收集进度四阶段进化：
  残缺玻璃 → 沙肤初成 → 晶骨流金 → 时之完体。
- **沙漏翻转**：按 `F` 让局部时间倒流——断桥沿裂痕重组、碎瓶重圆、
  沙魇的子弹原路返回反杀其主、全世界的沙粒逆流而上、天空浮现巨大沙漏投影。
- **记忆抉择**：10 粒记忆之沙，四种颜色（金·欢愉 / 蓝·离别 / 灰·恐惧 / 黑·罪悔）。
  每一粒都可**融入**（世界复苏 + 形态进化）或**埋葬**（身轻如风 + 移速提升）。
- **世界重铸**：随复苏进度，流沙从废墟表面退去露出砖石与黄铜的本色，
  枯树发芽、河流重淌、齿轮转动、幽灵 NPC 逐渐凝实并讲述生前的故事。
- **多结局**：完整之世 / 虚假天堂 / 永恒沙海 / 不完整的黎明——由你的抉择决定。

## 探索指引（剧透警告）

- 远处的彩色光柱即记忆之沙所在；
- 穹顶圣堂内的碎瓶、干涸河道上的断桥，都需要翻转沙漏修复；
- 灯塔顶端的记忆需要沿螺旋石阶跳跃攀登；
- 西南角的沉沙之窖埋着这场灾难的真相，有沙魇看守。

## 技术

纯前端原型，零构建：Three.js r147（UMD）+ UnrealBloom 后处理 + 自定义 Shader
（天空 / 河流）+ Canvas 程序化纹理 + WebAudio 程序化音效（无任何外部资源文件）。
